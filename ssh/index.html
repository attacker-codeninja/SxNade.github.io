# SSH File transfers



Now we will see that how we can copy folders using SCP

all we need to do is use additonal option of -r
which means recursive!

so for example we have a folder on our machine and we want to transfer this to the target machine we can do something as follows

scp -r folder/ 192.168.43.182:/root/Desktop


if we write this as [[ scp -r folder/* 192.168.43.182:/root/Desktop ]]
→ this will transfer all the files present in the folder but not the whole folder itself



similar will be to copy a folder from target machine to our machine//

scp -r root@192.168.43.182:/root/folder /root/Desktop

and this will Copy the folder from target machine to our Desktop!

or this also for only contents scp -r root@192.168.43.182:/root/folder/* /root/Desktop
